*HTML이란?
Hyper Text Markup Language
하이퍼텍스트는 링크로 연결된 텍스트 페이지
마크업 언어는 태그를 써서 문서의 구조를 정하거나 문서 내에 링크를 삽입할 때 사용되는 언어
결국, HTML은 웹에서 링크를 통해 이동 가능한 문서를 만드는 데 쓰이는 언어
사이트를 구성하는 웹페이지를 만드는 것

CSS로 웹페이지의 스타일을 만들어 디자인적인 요소를 추가

이벤트 생성은 자바 스크립트로 구현

*HTML 문서 만들기
메모장이나 문서파일을 열어 텍스트를 입력한 뒤 확장자를 .html로 바꿔준다.
메모장에서의 줄바꿈을 웹에서는 인식하지 못함
문단을 구부하려면 제목, 문단 구분을 지어줘야하는데 태그를 사용하여 구분해줌
시작태그와 종료태그로 짝을 맞춰 사용
시작태그 <>,종료태그</>안에 각 태그 이름을 쓰면됨
문단 구분은 paragraph의 약자인 'p'태그를 사용함
<p>, 그 문단 끝 뒤에는 </p>를 써줌

예)
<p>HTML 기초 강의</p>

<p>안녕하세요.</p>
<p>석준이와 함께 
HTML에 대해 알아보는 시간입니다.</p>

*HTML 문서 구조
html 문서는 크게 head와 body로 구분됨

head: 타이틀, 메타태그 , 스타일시트영역
body: 자바스크립트로 작성

head 영역은 브라우저에서 페이지를 열었을 때는 표시되지 않는 부분
타이틀은 브라우저 제목
메타태그는 해당 웹페이지를 설명하는 태그. 검색엔진이나 브라우저가 접근할 때 사용할 수 있는 정보를 담음
스타일시트는 해당 페이지를 꾸미기 위한 스타일(폰트,색상,배경 등) <style> 태그를 사용하여 정해줄 수도 있지만
주로 별도의 CSS파일을 만들어 참조

문서를 만들 때는 반드시 이 문서가 HTML 문서임을 선언해줘야 함
이때 사용하는 태그가 <!doctype html>
HTML 문서는 무조건 <!doctype html>로 시작해야함

HTML 문서의 시작은 <html>, 끝은 </html>을 써줘야함

헤드 정의는 <head> </head>안에 써야함
타이틀은 <title>테스트 페이지</title>을 <head> 태그 안에 넣어줌
브라우저의 탭에 표기되는 이름이 바뀜

바디 정의는 <body> </body>안에 써야함
본문 내용 중 문단 제목을 붙이기 위해서는 <h>태그를 사용
h1 ~ h6 숫자가 클수록 글자는 작아짐

본문 내용 입력은 문단 태그인 <p></p>를 사용
문단 내에서 줄바꿈은 <br>로 바꿔줌
